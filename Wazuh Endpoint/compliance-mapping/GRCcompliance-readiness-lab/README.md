# GRC Compliance Readiness Lab Using Wazuh

## Overview
This project demonstrates a practical Governance, Risk, and Compliance readiness assessment using Wazuh as the security monitoring and evidence collection platform.

The lab is based on a fictional healthtech startup named HopeHealth AI. The goal is to show how a security and compliance analyst can use Wazuh alerts, vulnerability findings, file integrity monitoring, and security configuration checks to support audit readiness.

## Frameworks Covered
This lab maps Wazuh evidence to common security and compliance frameworks, including:

- SOC 2 / Trust Services Criteria
- HIPAA Security Rule
- NIST 800-53
- GDPR
- ISO 27001 concepts
- HITRUST-style control thinking

## Tools Used
- Wazuh SIEM/XDR
- Wazuh Agent
- Linux endpoint
- Windows endpoint, optional
- Vulnerability Detection
- File Integrity Monitoring
- Security Configuration Assessment
- Authentication monitoring
- Incident response documentation

## What This Lab Demonstrates
- Compliance evidence collection
- Risk assessment
- Control mapping
- Gap analysis
- Remediation planning
- Audit readiness documentation
- Security monitoring with Wazuh

## Business Scenario
HopeHealth AI is a small healthtech startup preparing for customer security reviews and future compliance audits. The company needs to demonstrate that it has security monitoring, access control, vulnerability management, incident response, and evidence collection processes in place.

## Why Wazuh
Wazuh provides security monitoring capabilities that can support compliance activities, including:

- File Integrity Monitoring
- Security Configuration Assessment
- Vulnerability Detection
- Malware Detection
- Incident Response
- Compliance-tagged alerts for frameworks such as HIPAA, NIST 800-53, GDPR, PCI DSS, and Trust Services Criteria

## Key Deliverables
- Wazuh evidence documentation
- Compliance control mappings
- Risk register
- Gap analysis
- Remediation plan
- Auditor evidence package

This project aligns with IT Security and Compliance Analyst responsibilities such as:
